# Vendor Evaluation — Q2

Evaluating observability vendors to replace our in-house log pipeline by end of Q3.

## Finalists

1. **Relic** — hosted, mature, expensive.
2. **Foresight** — self-hostable, open-core, less polished.
3. **Quasar** — newer, cheapest, fewer integrations.

## Process

- Each vendor gets a 2-week POC.
- Technical eval + procurement review + legal review run in parallel.
- Decision by April 30.
