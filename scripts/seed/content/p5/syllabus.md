# CS 1410 — Syllabus (Spring)

## Course description

First-year introduction to computer science. Python, basic data structures, and computational thinking. No prior programming required.

## Learning outcomes

By the end of this course, students will be able to:
1. Read and write small Python programs using functions, conditionals, loops, and lists.
2. Decompose a problem into procedural steps.
3. Reason about time and space using informal Big-O language.
4. Use version control at a basic level (clone, commit, push).

## Grading

- Problem sets (10): 40%
- Two midterms: 30%
- Final project: 20%
- Participation: 10%

## Policies

- **Late**: 10% per day, up to 3 days. After that: 0 unless a dean's excuse is on file.
- **Collaboration**: discuss approaches, write your own code. Cite collaborators in comments.
- **AI tools**: allowed for explaining concepts; disallowed for writing problem-set solutions. When in doubt, ask.

## Weekly rhythm

- Mon: new concept (lecture).
- Wed: live coding + problem walkthrough.
- Fri: pair-programming lab with TAs.
