# Icebox

Ideas that aren't v0.1. Each line has a one-sentence reason for deferring.

- **iCloud sync** — needs Apple dev account + real crypto review, post-v0.1.
- **Widgets** — nice but requires iOS/Android app shell, not PWA-friendly.
- **Habit categories/tags** — wait for 3+ users to ask.
- **Streak recovery ("freeze days")** — gamification creep; revisit if users ask.
- **Sharing a habit with a friend** — social features are a different product.
- **Data export (CSV)** — add when someone actually wants it; localStorage is trivially exportable via devtools for now.
- **Dark mode** — looks easy but forces a color system decision; punt until v0.2.
