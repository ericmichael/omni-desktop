# Habit Tracker

My side project. A tiny habit tracker I actually want to use.

## Goals (v0.1)

- Add/edit/delete a habit.
- Check off a habit for today.
- See a streak count.
- Weekly reflection prompt on Sundays.

## Non-goals (yet)

- Multi-device sync.
- Social features.
- Gamification beyond streaks.
