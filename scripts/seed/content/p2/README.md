# habit-tracker

A small personal side project — track daily habits, see streaks, reflect weekly.

## Dev

```
npm install
npm run dev
```

## Build

```
npm run build
npm run preview
```

## Stack

- Vite + React 18 + TypeScript
- Tailwind (optional, wired up but light usage)
- No backend — localStorage for now; iCloud sync is on the icebox

## Status

Pre-v0.1 — see `v0.1 launch` milestone in the app.
